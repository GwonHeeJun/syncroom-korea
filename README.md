# SYNCROOM KOREA
SYNCROOM 방 참여 페이지의 한국 버전

```현재 SYNCROOM KOREA는 알파 테스트 중입니다. 모든 기능이 완전히 작동하지 않습니다.```

* * *
### 현재 작동하는 기능
> * 현재 열려있는 방 및 정보 표시
> * 각각의 방의 멤버 표시
> * 자동 방 새로고침
> * 비밀번호 없는 방 참여/임시 참여

### 현재 작동하지 않는 기능
> * 팝업 또는 별도의 입력창이 필요한 기능
> 
>     > 비밀번호 있는 방 참여, 방 링크 공유 등
> * 방 검색 / 필터 기능
> * 페이지 메뉴, 설정
